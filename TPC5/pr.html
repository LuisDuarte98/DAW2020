ABSTRACT:

O trabalho de casa número 5 consistia em criar um servidor node capaz de comunicar via http com um servidor json-server.

Primeiramente, inicializei o servidor json-server para que recebesse pedidos http e fosse capaz de ler o ficheiro json da escola de música e respondesse com o contéudo correspondente.

De seguida criei um servidor node que filtrasse os pedidos, e caso estes correspondam ao esperado, realiza-se pedidos ao json-server para obter o contéudo esperado. Para esta interligação foi usada a bibliotecaaxios.

Por último, criei este project record de forma a documentar o processo.
