ABSTRACT:

Para o terceiro trabalho para casa foi pedido que fosse criado um DTD que represente a estrutura de um ficheiro XML de arqueosítios previamente fornecido, e que de seguida fosse criado um XSLT para converter o documento num website funcional.

Face ao desafio proposto, comecei por modelar a estrutura do documento "arq.xml", recorrendo a expressões XLPath para perceber rapidamente a estrutura do documento, implementando-a depois num ficheiro DTD. De forma a comprovar a correção desta tarefa validei com sucesso o documento XML fornecido.

Posteriormente, realizei, com aquilo que aprendi na aula, um documento XSLT que gerasse com sucesso o website pretendido.

Já com os objetivos principais cumpridos, realizei este project record para servir de documentação

Por fim, decide criar um novo XSLT para estilizar este tipo de ficheiros XML (Project_Record´s). Já tinha criado algo semelhante no último trabalho de casa, pois tinha pesquisado na internet esta ferramenta. No entanto, depois de assistir à aula, percebi que a forma como tinha "atacado" anteriormente esta temática, estava longe de ser perfeita, e por isso decidi fazer um novo.
