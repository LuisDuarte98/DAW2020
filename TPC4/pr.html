ABSTRACT:

No trabalho de casa número 4 foi pedido que fosse criado um servidor node capaz de aceitar e responder a pedidos para o website de arqueossítios.

Primeiramente, criei um novo xsl de forma a gerar o website com um formato que permitisse criar um servidor capaz de aceitar pedidos através de urls do estilo: "localhost:7777/arqs/nome_ficheiro".

De seguida criei um servidor node que filtrasse os pedidos, e caso estes correspondam ao esperado, respondesse ao utilizador com a página correta.

Por último, usei o dts do project record feito na aula para gerar um xml para documentar o tpc, criando ainda um novo xsl capaz de transformar ficheiros válidos segundo o novo dtd.
